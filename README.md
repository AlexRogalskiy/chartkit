# @yandex-cloud/ChartKit

Library for rendering Yandex Charts scripts

## Install

```bash
npm i yandex-cloud/chartkit
```

## Usage

```jsx
import ChartKit, {ThemeProvider} from '@yandex-cloud/chartkit';

<ThemeProvider>
  <ChartKit id="1" type="yagr" data={data} lang="ru" />
</ThemeProvider>;
```
